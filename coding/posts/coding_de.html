Why coding is important in data engineering?

by Husein

When it comes to data engineering or data analysis, coding, besides statistics is extremely important for several reasons. First of all, coding allows us to work with much larger data sets than it's actually possible with spreadsheet apps like Excel or Google Sheets. The best example for this is vast amount of data sent through 2 billion tweets on Twitter on yearly basis. Another example is with Amazon that processes 1.1 billion transactions every year.


Processing that amount of data would not be possible without automation using programming languages like Python or R. Using one of the programming languages, we would automate and improve processes that previously might have to be completed manually and would take much more time.


Of course, Python by itself is not enough and we should know how to perform data analysis using SQL, CLI, Shell and other tools, but nevertheless it plays crucial role when it comes to automation.

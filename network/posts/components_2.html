Network components II

by Husein

Just to summarize, the whole purpose of the network is to get data from one place to another, which means that data will be moved through network somehow. Now what comprises our networks? We have many different network components, in this case I'll mention 8 most important, like clients, servers, hubs, WAP, switch, router, media and WAN link.


Client is a device used by an end-user to access the network with. It usually is something like PC, tablet, smartphone, smart TV or any other terminal device. Basically any device that connects to a network.


Server is a device that provides resources to the rest of the network. We have differenbtn servers that provide different functions like: web server, email server, chat server, print server and many more. These can be dedicated hardware or a specialized type of software that device uses.


Hub is an older piece of technology that connects network devices together (such as clients and servers). The problem with hubs was that they can lead to increased network errors, because hub receives information in one port and rebroadcasts it out all the other ports. For example, if we have 51 port hub, when one person is talking, the other 50 people have to listen to it too.

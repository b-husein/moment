Server basics I

by Husein

What is a web server? When I started learning about servers, one basic question that I've had was how hardware is different than the hardware of PCs that I've built and used in past. In this case, I think that I was asking the wrong question. From a hardware standpoint, anything with a connection can be a server, personal computer, laptop, Raspberry Pi, smartphone, any IoT device and similar.


Web server is not about hardware, though some hardware is better suited to run large web apps, instead of web server is basically just a block of code that serves web content. There are six basic things that server does in order to serve web content:


  1. LISTENS
  2. on a PORT
  3. for a REQUEST
  4. sent via TRANSPORT PROTOCOL
  5. returns a RESPONSE
  6. containing the requested RESOURCE
